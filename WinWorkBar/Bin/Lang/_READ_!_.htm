National language support

Each language file is a text file, that can be edited in notepad. Name of language file is it's language locale code identifier. Program at first run, searches for and loads language file with name of system default locale code. For example: if your default locale is Afrikaans, then program searches for and loads afk.txt language file if exists. If not, loads default eng.txt language file.

You can translate program to your locale. Simply edit any language file (for example eng.txt) to your language, and save it to name of your language locale code.
