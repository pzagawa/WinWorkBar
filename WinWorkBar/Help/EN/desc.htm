Windows for editing tasks/notes have possibility for calculating mathematic expressions and entering shortcuts for files, www or email addresses.

To proper shortcut work and display, it must be entered like in given examples:

  • http://test.com
    shortcut to internet page
  • mailto:test@test.com
    shortcut to email address
  • file:c:\windows\
    shortcut to folder on disk
  • file:c:\test.txt
    shortcut to file on disk

FunctionParser for mathematic expression calculating, works with following operators:

  • -, +, ^, *, /, %
  • & (logical product), | (logical sum)
  • =, <, >,
  • ( ), (brackets)

Most important functions:

  • abs(A) - absolute A value
  • sqrt(A) - square root of A
  • exp(A)
  • floor(A) - largest integer value less than A
  • ceil(A) - smallest integer value larger than A
  • int(A) - rounds A to closest integer number
  • if(A,B,C) - logical condition. If int(A) differs from 0 (true), returns B, else C
  • max(A,B) - if A>B, returns A, else B
  • min(A,B) - if A<B, retursn A, else B
  • acos(A), acosh(A)
  • asin(A), asinh(A)
  • atan(A), atan2(A,B), atanh(A)
  • cos(A), cosh(A), cot(A), csc(A)
  • log(A), log10(A)
  • sec(A)
  • sin(A), sinh(A)
  • tan(A), tanh(A)

Example expressions:

1+2 = 3

(4*10) / 10 - 100 = -96

-sin(sqrt(4^2+5^2)) = -0.1115231204

sqrt(2*3 + 4*5) = 5.099019514
