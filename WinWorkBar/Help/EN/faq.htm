Frequently asked questions

Where WinWorkBar stores data ?

Program can be used by many users on systems with multi-accounts feature (Windows 2000/XP/2003), also with applied restrictions. So, WinWorkBar determines itself best place for data store at first run. Usually it is user's home directory:

C:\Documents and settings\UserName\Local settings\Application data\Vetch Utilities\WinWorkBar\Data\

in systems that do not have this directory, data are stored in:

C:\Documents and settings\UserName\Application data\Vetch Utilities\WinWorkBar\Data\

alternatively, if system does not support multi-accounts, "Data" directory is placed in program directory.
