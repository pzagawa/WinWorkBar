Quick start

What is this WinWorkBar for ?

WinWorkBar stores user's tasks and notes and also reminds about terms and events. You can save your personal data and categorize them, save to html format, print from www browser, search and filter. Program reminds about tasks to do. You can easily and fast save any information in notes.

How to use this program ?

After clicking on WinWorkBar program icon on desktop, quick launch bar or WinWorkBar group in Start menu, main program window will appear - program panel, placed at right side of the screen. Main window contain (from the top):

  • menu bar

  • clock and date

  • calendar

  • tabs of today report, tasks list and notes list

Additionally, there is a program icon on tray notification area. After click on it with right mouse button, you have access to most important program options.

Creating new task

You can open new task window in many ways:

  • by clicking with left mouse button on "Tasks" item of report list, tab "Today"
  • by selecting menu item "New task" from "File" menu
  • by pressing "New" button on tab "Tasks"
  • by selecting menu item "New task" from tray icon popup menu
  • or by pressing keyboard shortcut (HotKey). Default Ctrl+Shift+F6

When new task window appears, simply write text in edit window. Beginning of this text will appear after save on tasks list ("Tasks" tab), on "Today" report list (by task date), and in small popup reminder window.

Task edit window contains following elements (from top):

  • categories select button
  • task term date, time, task term end date, button for editing repeats
  • priority select list, trackbar for setting task progress
  • task text edit field
  • buttons "Delete", "Save" and "Cancel"

After entering task text and pressing "Save" button (shortcut Ctrl+S), task will be saved with default settings. If you select categories for this task, it will be easy to filter, search and manage task in the future.

Before you save task, you can also select other date and time of term, or term date range. Task will be visible on "Today" report by entered dates. Additionally you can set repeats for showing tasks by any cycle.

Changing priority is useful for marking task on "Today" report list and also for filtering with "View" menu.

Trackbar for setting progress of task execution is useful for marking, filtering and searching tasks. Tasks finished (progress 100%), will not be shown anymore on "Today" report list after term date range.

Creating new note

Like before, you can open new note window in many ways:

  • by clicking with left mouse button on "Notes" item of report list, tab "Today"
  • by selecting menu item "New note" from "File" menu
  • by pressing "New" button on tab "Notes"
  • by selecting menu item "New note" from tray icon popup menu
  • or by pressing keyboard shortcut (HotKey). Default Ctrl+Shift+F7

When new note window appears, simply write text in edit window. Beginning of this text will appear after save on notes list ("Notes" tab), on "Today" report list (always, if date range "Show..." is not defined, or by given date range).

Note edit window contains following elements (from top):

  • categories select button
  • show date range (visibility)
  • note text edit field
  • buttons "Delete", "Save" and "Cancel"

After entering note text and pressing "Save" button (shortcut Ctrl+S), note will be saved with default settings.

If you select begin date "Show from", then note will be visible on "Today" report list from this date. If you select also "to" date, then note will be visible only in given date range.

"Toady" report list

Main program window contains three tabs. First of them, "Today", shows current tasks and notes for selected (default current) day. Tasks items are sorted from early to late hour. Notes are sorted by alphabetical order.

List is updated dependently from time and task state. If current time is equal with task time on list, then from left side of task item, small box appears, that shows expired time.

After task time text is task progress displayed in form of small bar with changing colors and fill, that depends of task progress selected with trackbar progress or by clicking "Finish" button in task reminder window.

After task progress bar is priority symbol displayed. For default priority of task it is line. For "important" task, it is red arrow, for "not important", it is blue arrow.

After these symbols is beginning of task text displayed.

Note is displayed only as a beginning of text.

Clicking on task/note item, opens it in edition mode.

"Tasks" and "Notes" lists

These lists displaying all tasks and notes items.

"View" menu settings changes filtering for these lists, by selected condition. All tasks and notes data are listed with details in list columns. After selecting one or more items, you can delete, view in browser or save it to html file.
