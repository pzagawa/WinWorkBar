Program requirements

WinWorkBar for proper work needs minimum system Windows 98. It works on Windows Millenium, Windows 2000 and Windows XP, too, and user interface has new, modern look only in XP system, which uses Windows Visual Styles technology.

In case of any problems with starting program on Windows 98 systems, you must update your system with newest version of Internet Explorer, that installs newest version of Windows Common Control and ShellApi library, too.

Minimum hardware configuration, that make it possible to run WinWorkBar program is a computer of class Pentium 100 with 32MB of RAM memory and as mentioned above, suitable Windows system.
