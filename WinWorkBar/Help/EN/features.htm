Program features

  • small size
  • fast operation
  • user friendly interface
  • handling data in XML format
  • shows tasks/notes report for every day
  • searching and filtering data
  • import and export data in XML format
  • export data to html format by default template
  • reminding about tasks in comfortable way with pop-up information window and selected sound
  • showing and hiding program window with keyboard shortcuts (HotKeys)
  • opening new tasks and notes with keyboard shortcuts (HotKeys)
  • assigning tasks and notes to selected categories
  • displaying tasks and notes in detail lists
  • clear and easy calendar
  • clock and date always visible on main window
  • most important options also accessible from tray icon menu
