What is this ?

WinWorkBar is a program for personal information management. It is program of a PIM (Personal Information Manager) type. You can write tasks to do or notes in text style with this program. Additionally there is calendar and clock.

WinWorkBar will help you to organize your current affairs. Will remind about important events and makes possible to fast save any information in tasks and notes with selected categories. There is also comfortable searching, tasks/notes preview and print in browser, export and import data in XML format, filtering by any condition, friendly information about tasks terms, keyboard shortcuts and even mathematical expressions calculating directly in task/note text. Most important informations are available in any time as nice report for current or selected day in main program window.

Text, that you are reading, is official and only version of program documentation, in electronical form.
