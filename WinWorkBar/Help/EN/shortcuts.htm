Keyboard shortcuts

Program functions can be invoked in two ways. With use of menu and user interface buttons, or with use of keyboard shortucts and single keys. Keyboard shortcuts are described as keys combination, for example:

Ctrl + P: simultaneous pressing Control key and P key

List of shortcuts

Ctrl + S saves data and closes active task/note window
ESC cancels changes and closes active task/note window
Ctrl + ENTER evaluates expression in current line. Works in task/note edit window
Ctrl + Shift + F10 default setting. Shows/hides program window
Ctrl + Shift + F6 default setting. Opens new task window
Ctrl + Shift + F7 default setting. Opens new note window
